# Design Context — Creator Engine (v2)

> Reinício completo — jun/2026. Direção anterior (risograph) descartada.

## Audience
- **Quem:** Rômulo, operação solo de personas digitais (Instagram, TikTok, YouTube, FanVue).
- **Contexto:** Uso diário prolongado — alterna execução (roteiros, calendário) e estratégia (analytics, funil, financeiro).

## Use Cases
1. **Operação:** criar/editar posts, calendário, status de contas, credenciais.
2. **Estratégia:** P&L, ROI por persona, discovery, ferramentas/SOPs/templates.

## Brand Personality
- **Tom:** Retro-futurista synthwave contido — painel de comando noturno, não arcade genérico.
- **Sensação:** Estação de controle dos anos 80 vista através de um CRT: fosforescente, técnico, íntimo.
- **Diferenciação:** Sidebar-instrumento compacta + camadas CRT (scanlines, vinheta, grid) com parcimônia; boot sequence no login.

## Aesthetic Direction

### Propósito
Central de comando para máquinas de conteúdo — deve parecer *hardware de interface*, não SaaS.

### Sub-estética: Synthwave contido
- Fundos sólidos violeta-profundo (sem gradientes roxo→azul decorativos).
- Fosforo primário: magenta elétrico (acentos, focus, nav ativa).
- Fosforo secundário: ciano só em hover/links — nunca em gradiente de hero.
- Status semânticos: verde (ok), âmbar (atenção), vermelho (crítico) — fosforescentes mas legíveis.

### Tipografia (par custom)
| Papel | Fonte | Uso |
|-------|-------|-----|
| Display | **Oxanium** | Títulos, logo, métricas hero |
| Body | **Sora** | UI, labels, parágrafos |
| Mono | **IBM Plex Mono** | Tabelas, KPIs, timestamps, dados densos |

### Layout
- **Sidebar compacta** (~56px): ícones fosforescentes; expande no hover (~220px) com labels.
- **Densidade equilibrada:** ritmo variado — grupos tight em tabelas, respiro entre seções.
- **Dark padrão**; tokens preparados para light toggle futuro (não implementar toggle agora).

### Efeitos CRT (camadas, todas sutis)
1. **Scanlines** — overlay global ~4% opacidade.
2. **Vinheta CRT** — bordas escurecidas, curvatura implícita.
3. **Glow fosforescente** — apenas display headings e métricas-chave (`text-shadow`, não box-shadow em tudo).
4. **Grid técnico** — retícula 24px no fundo do main, quase invisível.
5. **Boot sequence** — animação de login estilo terminal (typewriter + cursor).

### Motion
- Entrada staggered no dashboard (fade + translateY, ease-out-quart).
- Sidebar expand com `transform` + opacity, não width animado.
- Respeita `prefers-reduced-motion`.

## Constraints
- Next.js 16, Tailwind v4 sem compiler de cores → tokens CSS OKLCH + `var(--*)`.
- pt-BR, acessibilidade (focus-visible, contraste WCAG AA em texto body).
- Não quebrar rotas/APIs existentes — redesign é shell + dashboard primeiro.

## Don't
- Gradientes roxo-azul em heroes/cards (fingerprint AI slop).
- Inter, Roboto, glassmorphism decorativo, cards idênticos em grid infinito.
- Glow em tudo — parcimônia é o diferencial do "contido".
- Herdar risograph, Syne, magenta riso, kickers laranja xerocado.

## Implementação — ordem sugerida
1. `globals.css` — tokens OKLCH synthwave + camadas CRT + fontes.
2. `layout.tsx` root — carregar Oxanium, Sora, IBM Plex Mono.
3. `sidebar.tsx` — compacta + hover expand.
4. `login/page.tsx` — boot sequence.
5. Dashboard (`page.tsx`, `stats.tsx`, `personas-table.tsx`) — primeiro ecrã polido.
